# HashiCorp Cloud Platform (hcp)

HashiCorp Cloud Platform (HCP) is a fully managed platform for HashiCorp
products including Vault, Consul, Packer, Boundary, Waypoint, and Terraform.
HCP provides a unified set of APIs for managing infrastructure, secrets,
service networking, and image pipelines across cloud and on-premises
environments.

The HCP Vault Secrets API enables programmatic management of applications,
secrets, and integrations within HashiCorp Cloud Platform Vault Secrets, a
